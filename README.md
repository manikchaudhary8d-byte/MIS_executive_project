# 📊 MIS Executive Project – Excel Dashboard

## 📌 Project Overview
This project demonstrates the creation of an Excel-based MIS (Management Information System) dashboard to analyze business data and generate actionable insights.  
It focuses on data cleaning, reporting, and visualization using core Excel functions.

---

## 🎯 Objectives
- Analyze and monitor business data efficiently  
- Create an interactive dashboard for reporting  
- Automate calculations using Excel functions  
- Support data-driven decision-making  

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel  

---

## 📂 Dataset Details
The dataset includes structured business data such as:
- Employee / Customer Name  
- Product / Category  
- Sales / Performance metrics  
- Quantity / Transactions  

---

## ⚙️ Key Features & Functions Used
- **COUNTIF** – Used for conditional counting  
- **SUMIF** – Used for conditional summation  
- **VLOOKUP** – Used for data retrieval and merging datasets  
- **FILTER** – Used for dynamic data extraction  
- **Pivot Tables** – Used for summarizing large datasets  
- **Dashboard Creation** – Visual representation of key insights  

---

## 📊 Dashboard Features
- Summary of key metrics (Total Sales, Count, etc.)  
- Category-wise and product-wise analysis  
- Dynamic filtering for better insights  
- Clean and structured layout for easy understanding  

---

## 🧠 Key Insights
- Identified top-performing categories 
- Analyzed trends in sales   
- Improved visibility of business performance through dashboard  
---

## 🚀 How to Use
1. Open the Excel file  
2. Navigate to the dashboard sheet  
3. Use filters and visuals to explore insights  

---

## 💡 Learning Outcomes
- Practical use of Excel functions in real scenarios  
- Data cleaning and preparation  
- Building MIS reports and dashboards  
- Converting raw data into meaningful insights
